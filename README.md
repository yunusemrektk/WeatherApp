# WeatherApp

One of the main Purpose of this repo is to practice of Clean Architecture and MVVM usages. 
Location Tracker is used in this project to get current locations' weather forecasts.

- Clean Architecture
- MVVM
- Dependency Injection
- Jetpack Compose
- Dagger Hilt
- Retrofit
